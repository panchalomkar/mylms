ABOUT

This plugin for Moodle allowing a teacher to manually grade their participant on essay type questions.

The plugin allow a teacher or mentor who are the part of the course and group.
In case the course has some assessments and assessments have the essay type questions. 
This plugin basically allow for teacher to manually grade the question based on their responses.
Only two mentor can grade for a single user and average grade will be consider for a final assessment grade.

INSTALLATION

Just place the question_grade directory inside your Moodle's local directory.
Install the plugin and browse to:

Site Administration->Plugins->Local plugins->Question Grade
